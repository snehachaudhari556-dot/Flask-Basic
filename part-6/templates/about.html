About This App

My To-Do App is a simple task management application built using Flask. It helps users organize daily tasks by allowing them to add, view, and delete tasks easily in a clean and user-friendly interface.

This project was created as part of a learning journey to understand Flask fundamentals such as routing, template inheritance, Jinja2 templating, and responsive design. It works smoothly on both desktop and mobile devices.
